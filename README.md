## The Repository

This repository contains starter code from an existing text editor web application and new code implementing IndexedDB so the application can effectively function as a Progressive Web Application. 

This functionality allows the user to create notes or code snippets with or without an internet connection. Starter code was provided by Columbia Coding Bootcamp.

---

## Installation

The code may be accessed by visiting the [GitHub repository](https://github.com/choelzli/text-editor-pwa) or visiting the site [URL](https://express-text-editor-pwa-ch-[update].herokuapp.com/).

Clone the repo:
```
git clone https://github.com/choelzli/text-editor-pwa.git
```
---

## Usage

After installing the required dependencies, type the following command in the Terminal:
```
npm run start
```
Then access in Thunder Client and type http://localhost:3000/
